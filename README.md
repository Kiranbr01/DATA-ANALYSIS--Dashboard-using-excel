# **Sales Data Analysis Project**


## **1. Project Overview**
The **Sales Data Analysis Project** is developed to automate, analyze, and visualize sales performance data using Microsoft Excel.  
The repository contains a macro-enabled workbook named **`salesdata2.xlsm`**, designed to simplify data analysis through automation, pivot-based summaries, and dashboard visualization.

This project highlights the application of Excel’s advanced functionalities—such as VBA macros, pivot tables, and charts—in transforming raw data into actionable business insights
