# **Sales Data Analysis Project**


## **1. Project Overview**
The **Sales Data Analysis Project** is developed to automate, analyze, and visualize sales performance data using Microsoft Excel.  
The repository contains a macro-enabled workbook named **`salesdata2.xlsm`**, designed to simplify data analysis through automation, pivot-based summaries, and dashboard visualization.

This project highlights the application of Excel’s advanced functionalities—such as VBA macros, pivot tables, and charts—in transforming raw data into actionable business insights

## **2. Project Objectives**
The primary objectives of this project are:

1. To automate sales data entry, cleaning, and analysis using Excel macros (VBA).  
2. To summarize sales data by product, region, and time period.  
3. To identify trends and performance variations through dashboards and charts.  
4. To measure and evaluate key performance indicators (KPIs) for business growth.  
5. To demonstrate Excel as an effective analytical tool for data-driven decision-making.
